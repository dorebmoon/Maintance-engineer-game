# ⚔️ TechQuest - 技术探索之旅

一个交互式闯关游戏网站，帮助你循序渐进地掌握 **SQL**、**Redis**、**Kafka** 和 **ELK** 四大后端核心技术。

🎮 **纯前端项目，无需后端，打开 `index.html` 即可开始冒险！**

---

## 🚀 快速开始

```bash
# 方式一：直接用浏览器打开
open index.html

# 方式二：使用本地服务器（推荐）
python3 -m http.server 8080
# 然后访问 http://localhost:8080

# 方式三：使用 Node.js
npx serve .
```

## 🌍 四大冒险世界

### 🗃️ SQL 数据库（12 关）

从零开始掌握 SQL 查询语言，循序渐进。

| 难度 | 关卡 | 学习内容 |
|------|------|----------|
| ⭐ 初级 | 1-4 | SELECT、WHERE、AND/OR 多条件查询 |
| ⭐⭐ 中级 | 5-7 | ORDER BY 排序、GROUP BY 聚合、HAVING 过滤 |
| ⭐⭐⭐ 高级 | 8-9 | INNER JOIN、LEFT JOIN 表连接 |
| ⭐⭐⭐⭐ 专家 | 10-12 | 子查询、窗口函数 ROW_NUMBER、CTE |

### 🔴 Redis 缓存（11 关）

学习 Redis 内存数据库的核心操作与高级应用。

| 难度 | 关卡 | 学习内容 |
|------|------|----------|
| ⭐ 初级 | 1-4 | SET/GET、过期时间、Hash 哈希、List 列表 |
| ⭐⭐ 中级 | 5-7 | Set 集合、Sorted Set 有序集合、INCR 原子计数 |
| ⭐⭐⭐ 高级 | 8-9 | MULTI 事务、PUB/SUB 发布订阅 |
| ⭐⭐⭐⭐ 专家 | 10-11 | Pipeline 批量操作、缓存穿透防护 |

### 📨 Kafka 消息队列（11 关）

掌握 Apache Kafka 分布式消息系统的核心概念。

| 难度 | 关卡 | 学习内容 |
|------|------|----------|
| ⭐ 初级 | 1-4 | 创建 Topic、生产消息、消费消息、查看 Topic |
| ⭐⭐ 中级 | 5-7 | 消费者组、偏移量管理、自定义分区策略 |
| ⭐⭐⭐ 高级 | 8-9 | Rebalance 与消费者配置、Exactly-Once 语义 |
| ⭐⭐⭐⭐ 专家 | 10-11 | Kafka Streams 流处理、Kafka Connect 连接器 |

### 📊 ELK 日志系统（11 关）

学习 Elasticsearch + Logstash + Kibana 日志系统。

| 难度 | 关卡 | 学习内容 |
|------|------|----------|
| ⭐ 初级 | 1-4 | ES 索引、索引文档、Match 查询、Term 查询 |
| ⭐⭐ 中级 | 5-7 | Bool 组合查询、Range 范围查询、聚合分析 |
| ⭐⭐⭐ 高级 | 8-9 | Logstash Input 配置、Grok 日志解析 |
| ⭐⭐⭐⭐ 专家 | 10-11 | Kibana KQL 查询、ELK 架构设计 |

## 🎮 游戏特性

- **⭐ 经验值系统** — 每个关卡有对应的 XP 奖励，难度越高奖励越多
- **🔓 闯关解锁** — 必须通过当前关卡才能解锁下一关，确保循序渐进
- **🏅 成就系统** — 13 个成就等你解锁（首次通关、世界大师、XP 里程碑等）
- **🔥 学习打卡** — 连续学习天数追踪，激励每天学习
- **💡 提示系统** — 卡关时可查看提示和知识讲解
- **💾 自动存档** — 进度和代码自动保存到浏览器本地存储
- **🎨 暗色主题** — 赛博朋克风格 UI，护眼舒适
- **📱 响应式设计** — 支持手机、平板和桌面端
